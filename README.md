A modern and responsive **Healthcare Dashboard UI** built to manage patient data, visualize diagnosis history, and monitor vital signs in a clean and intuitive interface.

---

## 🚀 Project Overview

This project is a frontend implementation of a healthcare management system dashboard. It helps healthcare professionals easily view patient information, track medical history, and analyze health metrics using interactive charts.

The goal of this project is to demonstrate **real-world UI design and frontend development skills**.

---

## ✨ Features

* 👨‍⚕️ Patient Management (list & details view)
* 📊 Diagnosis History with visual charts
* ❤️ Vital Signs Monitoring
* 🔍 Patient Search functionality
* 📱 Fully Responsive Design
* 🎯 Clean and modern dashboard UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Chart.js
